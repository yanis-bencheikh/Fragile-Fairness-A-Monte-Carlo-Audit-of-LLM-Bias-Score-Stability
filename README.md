# Fragile-Fairness-A-Monte-Carlo-Audit-of-LLM-Bias-Score-Stability
Monte Carlo framework auditing the robustness of LLM stereotype bias scores (Stereo Win Rate) under 4 cumulative noise doses. Benchmarks BLOOMz, Gemma, Llama 3.1, Mistral on CrowS-Pairs EN/FR and StereoSet. Bias-word protection, Z-score non-robustness statistics, multilingual analysis. Python + R.
